# Personal Portfolio

A personal portfolio website built as part of the [roadmap.sh](https://roadmap.sh) frontend development learning path.

## Overview

This project demonstrates core web development skills including HTML, CSS, and JavaScript to create a professional portfolio showcasing projects and skills.

## Features

- Responsive design
- Project showcase
- Contact section
- Modern UI/UX

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize with your own content

## Technologies

- HTML5
- CSS3
- JavaScript
